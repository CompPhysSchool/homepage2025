## デプロイ前の動作確認

ローカルサーバーを立ち上げて，HPの動作確認を行います．やり方の一例を以下に示します．

1. brew や npm 等を用いて http-server をインストールする．
2. homepage2025 のルートディレクトリに移動する．
3. `http-server` を実行する．
4. ブラウザで `http://192.168.53.21:8080` にアクセスする．
